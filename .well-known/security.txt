# Reporting a Vulnerability
#
# For non-sensitive security issues, please create an issue at:
# https://github.com/motowilliams/bolt/issues
#
# For sensitive security vulnerabilities that should not be disclosed publicly,
# please use GitHub's private security advisory reporting if available, or
# contact @motowilliams directly through GitHub to arrange private disclosure.
#
# You should receive a response within 48 hours. If you do not receive a
# response, please follow up via GitHub mentioning @motowilliams.
#
# Please include in your report:
# - Description of the vulnerability
# - Steps to reproduce
# - Potential impact assessment
# - Suggested fix (if available)
#
# See our full Security Policy at the Policy URL above for:
# - Response timelines and SLAs
# - Coordinated disclosure process
# - Scope of security coverage
# - Known limitations and acceptable risks
